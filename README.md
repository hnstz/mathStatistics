# Математическая статистика: Лабораторные работы

Данный репозиторий содержит цикл лабораторных работ, посвященных вычислительным методам математической статистики и анализу случайных величин. В ходе работ исследуются свойства классических распределений, методы визуализации данных и оценки статистических характеристик.

## Технологии
Проект реализован на языке Python с использованием следующих библиотек:
* **NumPy** — для генерации выборок и проведения матричных вычислений.
* **SciPy (stats)** — для работы с теоретическими распределениями и статистическими критериями.
* **Matplotlib** — для построения гистограмм, графиков плотности, боксплотов и эмпирических функций распределения.

## Структура репозитория
Репозиторий организован по принципу ветвления. Ветка `main` содержит только общую документацию. Каждая лабораторная работа находится в своей ветке:

* `lab-1` — Исследование плотности распределения и гистограмм.
* `lab-2` — Оценка характеристик положения и рассеяния.
* `lab-3` — Визуализация данных с помощью диаграмм размаха (boxplot) и поиск выбросов.
* `lab-4` — Построение и анализ эмпирической функции распределения (ECDF).

## Описание работ

### Лабораторная работа 1-2
Сравнительный анализ теоретических и эмпирических распределений (Нормальное, Коши, Лапласа, Пуассона, Равномерное). Реализовано построение ядерных оценок плотности и расчет оптимального количества интервалов (bins) для гистограмм.

### Лабораторная работа 3
Анализ выборок на наличие выбросов с использованием правила межквартильного размаха (IQR). Построение графиков Boxplot для различных объемов выборок (n=20, n=100).

### Лабораторная работа 4
Сравнение эмпирической функции распределения с теоретической. Реализована визуализация ступенчатых графиков ECDF для анализа сходимости оценок при увеличении объема данных.

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone [https://github.com/username/repository-name.git](https://github.com/username/repository-name.git)
```

2. Перейдите в ветку нужной лабораторной работы:
```bash
git checkout lab-1
```

3. Установите необходимые зависимости:
```bash
pip install numpy scipy matplotlib
```

4. Запустите скрипт:
```bash
python lab1.py
```

## Требования к среде
* **Python 3.8+**
* **Поддержка вывода графических окон**: наличие `Tkinter` или других бэкендов (например, PyQt) для корректного интерактивного отображения графиков `Matplotlib`.
